## 📖 Project Overview

This project focuses on **forecasting energy consumption using Time Series AI techniques**.  
Accurate energy prediction is essential for **smart grids, energy optimization, and sustainable resource management**.

The project implements both **statistical and deep learning models** to analyze temporal patterns and generate accurate forecasts.

---

## 🎯 Objectives

- Analyze time series energy data  
- Detect trends, seasonality, and patterns  
- Build forecasting models  
- Compare classical vs deep learning approaches  
- Evaluate model performance  
- Generate future energy consumption predictions  

---

## 📊 Dataset

- 📂 Time Series Energy Consumption Dataset  
- ⏱️ Frequency: Hourly/Daily  
- 📈 Features:
  - Energy usage (target)
  - Time-based features (hour, day, month)
  - External variables (optional)
